# Example

To run this example:

- Clone the w3ui repository and enter the `w3ui` directory

  ```sh
  git clone https://github.com/web3-storage/w3ui
  cd w3ui
  ```

- Change to this example directory and install dependencies

  ```sh
  cd examples/vanilla/multi-file-upload
  npm install
  ```

- Run the example

  ```sh
  npm start
  ```
